# Referencie

## Page Purpose

Ukázať referenčné projekty Studio Terra Verde prostredníctvom kvalitných fotografií a krátkych popisov.

## Content Direction

Fotografie referenčných projektov s krátkym popisom.

## Recommended Intro Copy

Každý projekt vzniká individuálne podľa charakteru miesta, architektúry a predstáv klienta. V našich realizáciách spájame premyslený dizajn, kvalitné materiály, precízne remeselné spracovanie a dlhodobú starostlivosť o zeleň.

## Recommended Project Card Content

Each reference should include:

- Project title.
- Project type.
- Location if available.
- Short description.
- Main image.
- Optional gallery.
